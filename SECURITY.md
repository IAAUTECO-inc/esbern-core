# Security Policy

## Supported Versions
We provide security updates for the following versions of `esbern-core`:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
As an initiative focused on cybersecurity and sovereign infrastructure, we take security very seriously.

**Please do not report security vulnerabilities through public GitHub issues.**

If you believe you have found a security vulnerability, please send an encrypted email to **contact@iaauteco.ovh**. 

### Our Process
1. Acknowledge receipt of the report within 48 hours.
2. Provide an estimated timeframe for a fix.
3. Notify the community once a patch is available, crediting the researcher if desired.

We adhere to the principles of coordinated disclosure and appreciate your efforts to keep our sovereign AI infrastructures secure.
